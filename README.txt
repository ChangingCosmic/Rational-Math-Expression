This program creates a Rational struct and performs mathematical operations on two rational numbers.

To compile the program, run the command:
    @gcc -g -Wall rational.o test_rational.o -o rational_math -lm
or using the makefile command:
    @make

To run the program, run the command:
    @./rational_math testStarter.txt testStarterOutput.txt
where 'input1.txt' is the file to read in the rational numbers and 'output.txt' is the file to compare the answer to
